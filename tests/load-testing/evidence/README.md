# Bằng chứng load testing

Thư mục này dùng để lưu bằng chứng có thể commit vào repo.

- `results/` là nơi sinh kết quả thật khi chạy máy local và có thể bị bỏ qua bởi `.gitignore`.
- `evidence/` dùng để lưu file mẫu, bảng so sánh hoặc trích xuất kết quả cần nộp kèm báo cáo.

Khi chạy xong test, có thể copy file `.txt` hoặc `.md` quan trọng từ `results/` sang thư mục này để giữ bằng chứng trong repo.
